# 登录指南 / Login Guide

## 🚀 快速开始

由于这是一个纯前端应用（已删除后端），系统默认使用**模拟模式（Mock Mode）**，您可以直接使用以下账号登录：

### 默认演示账号

**邮箱**: `demo@example.com`  
**密码**: `demo123`

### 登录步骤

1. 启动前端开发服务器：
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. 访问 http://localhost:3000

3. 在登录页面输入：
   - Email: `demo@example.com`
   - Password: `demo123`

4. 点击 "Login" 按钮即可登录

## 📝 注册新账号

您也可以注册新账号：

1. 点击登录页面的 "Register here" 链接
2. 填写邮箱和密码（至少6个字符）
3. 可选填写姓名
4. 点击 "Register" 完成注册
5. 注册后会自动登录

## 🔧 配置说明

### 使用模拟模式（默认）

系统默认使用模拟数据，所有数据存储在浏览器的 `localStorage` 中：

- 登录/注册信息保存在本地
- 公司数据保存在本地
- 财务分析数据是模拟生成的

### 连接真实后端

如果您有后端API，可以修改 `.env` 文件：

```env
VITE_API_BASE_URL=http://localhost:8000
VITE_USE_MOCK=false
```

## 💡 功能说明

### 模拟模式下的功能

- ✅ 用户注册和登录
- ✅ 创建和管理公司
- ✅ 查看财务仪表盘（模拟数据）
- ✅ 上传文件（模拟上传）
- ✅ 查看报表和图表（模拟数据）

### 数据持久化

在模拟模式下，所有数据保存在浏览器的 `localStorage` 中：
- 用户账号信息
- 公司列表
- 登录状态

**注意**: 清除浏览器缓存会删除所有本地数据。

## 🎯 使用提示

1. **首次使用**: 使用默认账号 `demo@example.com` / `demo123` 登录
2. **创建公司**: 登录后进入 Settings 页面创建公司
3. **查看数据**: Dashboard 和 Reports 页面会显示模拟的财务数据
4. **上传文件**: Upload 页面可以模拟上传文件（不会真正处理）

## ❓ 常见问题

**Q: 登录失败怎么办？**  
A: 确保使用正确的邮箱和密码，或注册新账号。

**Q: 数据丢失了？**  
A: 检查是否清除了浏览器缓存，模拟模式下数据存储在 localStorage。

**Q: 如何重置？**  
A: 在浏览器控制台运行：`localStorage.clear()` 然后刷新页面。
